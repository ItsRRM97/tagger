Penn Treebank Parts of Tags

Usage

penn_tags(print = TRUE)

Arguments

print
logical. If TRUE a left justified pretty print version of the data.frame is printed.

Value

Invisibly returns a data frame of tags and meaning.

Description

A list of Penn Treebank parts of tags and their meaning.

penn_tags()
