Combine Words + Tags

Usage

as_tags(x, sep = "/", word.first = TRUE, ...)

Arguments

x
A tag_pos object or a named list of vectors.
sep
A separator to put between the word and the tag.
word.first
logical. If TRUE the word comes before the tag.
...
ignored.

Value

Returns a combined character vector of words and tags.

Description

A traditional style of displaying words and tags in the same vector.

Examples

(x <- tag_pos("I need $54 to go to the movies."))


Not using parallel process to tag.

[1] "I/PRP need/VBP $/$ 54/CD to/TO go/VB to/TO the/DT movies/NNS ./."



c(x) ## The true structure of a `tag_pos` object



[[1]]
     PRP      VBP        $       CD       TO       VB       TO       DT      NNS        . 
     "I"   "need"      "$"     "54"     "to"     "go"     "to"    "the" "movies"      "." 




as_tags(c(x))



[1] "I/PRP need/VBP $/$ 54/CD to/TO go/VB to/TO the/DT movies/NNS ./."



as_tags(c(x), word.first=FALSE)



[1] "PRP/I VBP/need $/$ CD/54 TO/to VB/go TO/to DT/the NNS/movies ./."



as_tags(c(x), sep="|")



[1] "I|PRP need|VBP $|$ 54|CD to|TO go|VB to|TO the|DT movies|NNS .|."



gsub("([^ ]+)", "(\\1)", as_tags(c(x), sep="="))



[1] "(I=PRP) (need=VBP) ($=$) (54=CD) (to=TO) (go=VB) (to=TO) (the=DT) (movies=NNS) (.=.)"
