Counts of Part of Speech Tags by Grouping Variable(s)

Usage

count_pos(x, grouping.var = NULL, group.names, pretty = TRUE, ...)

Arguments

x
A tag_pos object or a named list of vectors.
grouping.var
The grouping variable(s). Default NULL generates one row for each element in x.
group.names
A vector of names that corresponds to group.var. Generally for internal use.
pretty
logical. If TRUE pretty printing is used. Pretty printing can be turned off globally by setting options(tagger_pretty = FALSE).
...
ignored

Value

Returns a tbl_df of counts of parts of speech.

Description

Tabulate counts of part of speech tags by grouping variable(s).

Examples

data(presidential_debates_2012_pos) # pre tagged data
count_pos(presidential_debates_2012_pos)
